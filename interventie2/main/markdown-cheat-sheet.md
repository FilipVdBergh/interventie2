# Headings
Je kan drie niveaus gebruiken om je tekst in te delen:
# Hoofdstukken
## Paragrafen
### Subparagrafen

# Lijsten
## Genummerde lijst
1. First item
2. Second item
3. Third item

## Ongenummerde lijst
- First item
- Second item
- Third item

# Opmaak
- Je kan tekst **vet** en *schuin* afdrukken.
